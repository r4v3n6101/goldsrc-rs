# goldsrc-rs

Collection of file's parsers for goldsrc engine.

# Supported files

- [x] **.wad** containing fonts, mip textures, simple pictures
- [x] **.bsp** with all lumps support
- [ ] **.spr**
- [ ] **.mdl**

## Installation

In your **Cargo.toml** add new dependency:

```toml
[dependcies]
goldsrc-rs = "1"
```
